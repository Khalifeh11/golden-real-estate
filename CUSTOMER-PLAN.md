# Golden Land Real Estate — Website Project Plan

---

## 1. Project Overview

We are building a **modern, professionally designed website** for Golden Land Real Estate to replace your current website. The new website will be faster, mobile-friendly, and easier for your visitors to browse properties.

**All of your existing data will be transferred** to the new website — nothing will be lost. Every property listing, agent profile, photo, and contact submission you have today will carry over seamlessly.

---

## 2. Website Pages


| Page                   | What Visitors See                                                                  |
| ---------------------- | ---------------------------------------------------------------------------------- |
| **Home**              | A welcoming landing page with a hero banner, featured properties, and quick search |
| **About Us**          | Your company story, mission, and values                                            |
| **Properties**        | Full property listings with search bar, filters, and grid/list view                |
| **Property Detail**   | Individual property page with photos, specs, description, and contact form         |
| **Agents**            | A directory of all your agents with photo and contact info                          |
| **Agent Profile**     | An agent's bio and the properties they represent                                   |
| **Admin Login**       | Private login page for administrators only                                         |
| **Admin Dashboard**   | Your control center — see website activity at a glance                             |
| **Manage Properties** | Add, edit, or remove any property listing; mark as featured                        |
| **Manage Agents**     | Add or update agent profiles and photos                                            |
| **Contact Inbox**     | View all visitor inquiries received through the website                            |


---

## 3. Property Search & Filters

Your visitors will be able to find properties quickly using the following tools:

| Filter               | Options                                                          |
| -------------------- | ---------------------------------------------------------------- |
| **Keyword**          | Free-text search (e.g., "sea view", "pool")                     |
| **Reference Number** | Exact match by property reference number                         |
| **Category**         | For Sale, For Rent, Sold                                         |
| **Country**          | Select a country                                                 |
| **District**         | Select a district within the chosen country                      |
| **City**             | Select a city within the chosen district                         |
| **Price Range**      | Adjustable slider from minimum to maximum price                  |
| **Space Range**      | Adjustable slider from minimum to maximum area (in square meters)|


---

## 4. Property Listing Details

Each property page will display:

- **Photo carousel** — swipeable image gallery with full-screen viewing
- **Status** — For Sale, For Rent, or Sold
- **Area** — size in square meters
- **Bedrooms** — number of bedrooms
- **Bathrooms** — number of bathrooms
- **Parking** — number of parking spots
- **Reference number** — your existing reference numbers are preserved (e.g., #225520)
- **Description** — free-text field with full property details (rooms, finishes, amenities, floor, building age, etc.)
- **Price** — amount and currency (e.g., USD 1,900/month), with payment terms
- **Location** — country, district, and city
- **Assigned agent** — name, photo, WhatsApp link, email, phone number, and short bio
- **Contact form** — visitors can send an inquiry directly from the property page
- **Property search** — search for other properties directly from the listing page
- **Recommended properties** — a section showing related properties for the visitor to browse

---

## 5. Admin Dashboard

As the website owner, you will have access to a **private control panel** that only administrators can see. This is where you manage everything behind the scenes:

- **Website overview** — see your total properties, agents, and recent inquiries at a glance
- **Add and manage properties** — create new listings, edit any existing listing, mark properties as featured, or remove them
- **Manage agents** — add new agent profiles, update their information and photos, or remove agents who are no longer active
- **Contact inbox** — view every inquiry visitors have submitted through the website, and keep track of which ones you have responded to

---

## 6. Additional Features

- **WhatsApp button** — a floating WhatsApp icon on every page, linking directly to your number so visitors can reach you instantly
- **Social media links** — Instagram and Facebook icons in the website footer
- **Mobile-friendly design** — the website will look and work great on phones, tablets, and desktop computers
- **Contact form with notifications** — every inquiry submitted through the website is sent to you by email
- **Search engine optimized** — your properties and pages are structured so they appear in Google and other search engines

## 10. Proposed Color Palette

We reviewed the colors currently used on your website and are recommending a refined palette that feels more polished and premium — fitting for a real estate brand.

### Current Website Colors

| Color       | Hex       | Used For                      |
| ----------- | --------- | ----------------------------- |
| Dark Gray   | `#2B3A42` | Navigation, footer, sidebar   |
| Cyan Blue   | `#00D4FF` | Links, headings, logo         |
| White       | `#FFFFFF` | Backgrounds, text             |
| Orange      | `#FFA503` | Prices                        |
| Light Cyan  | `#5BC0DE` | Search bar, underlines        |
| Mid Gray    | `#888888` | Body text, labels             |

### Recommended Palette

| Color       | Hex       | Used For                               |
| ----------- | --------- | -------------------------------------- |
| Charcoal    | `#3D4F5F` | Navigation, headers                    |
| Deep Navy   | `#1A2A35` | Footer, dark sections                  |
| Muted Cyan  | `#00B8D9` | Links, icons, accents                  |
| Gold        | `#C9A84C` | Call-to-action buttons, prices, badges |
| Warm White  | `#F5F5F0` | Content backgrounds                    |
| Slate       | `#5A6E7F` | Secondary text, borders                |
| Dark Slate  | `#2C3E4A` | Sidebar, card overlays                 |

### What We Changed and Why

| Change                    | Before → After          | Reason                                                         |
| ------------------------- | ----------------------- | -------------------------------------------------------------- |
| Cyan toned down           | `#00D4FF` → `#00B8D9`  | Less neon, more professional                                   |
| Orange replaced with gold | `#FFA503` → `#C9A84C`  | Premium, luxury feel — brings back the gold from your old site |
| Gray warmed up            | `#2B3A42` → `#3D4F5F`  | Less cold and industrial                                       |
| Pure white to warm white  | `#FFFFFF` → `#F5F5F0`  | Softer and easier on the eyes                                  |

These colors are a starting point — we are happy to adjust based on your feedback.
