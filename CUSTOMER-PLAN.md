# Golden Land Real Estate — Website Project Plan

---

## 1. Project Overview

We are building a **modern, professionally designed website** for Golden Land Real Estate to replace your current website. The new website will be faster, mobile-friendly, and easier for your visitors to browse properties.

**All of your existing data will be transferred** to the new website — nothing will be lost. Every property listing, agent profile, photo, and contact submission you have today will carry over seamlessly.

---

## 2. Website Pages

| Page | What Visitors See |
|------|-------------------|
| **Home** | A welcoming landing page with a hero banner, featured properties, and quick search |
| **About Us** | Your company story, mission, and values |
| **Properties** | Full property listings with search bar, filters, and grid/list view |
| **Property Detail** | Individual property page with photos, specs, description, and contact form |
| **Agents** | A directory of all your agents with photo and contact info |
| **Agent Profile** | An agent's bio and the properties they represent |
| **Login / Register** | Visitors can create an account using email or Google sign-in |
| **Dashboard** | A personal area for registered users to manage their activity |
| **Submit Property** | Registered users can submit a new property listing for review |
| **My Properties** | Users can view, edit, or remove their own submitted listings |
| **Favorites** | Users can save properties they like and revisit them anytime |
| **Admin Dashboard** | Your control center — see website activity at a glance |
| **Manage Properties** | Add, edit, or remove any property listing; mark as featured |
| **Review Submissions** | Approve or decline properties submitted by registered users |
| **Manage Agents** | Add or update agent profiles and photos |
| **Contact Inbox** | View all visitor inquiries received through the website |
| **Manage Users** | View registered users and assign roles |

---

## 3. Property Search & Filters

Your visitors will be able to find properties quickly using the following tools:

**Search by:**
- Location or city name
- Keyword (e.g., "sea view", "pool")
- Reference number (exact match)

**Filter by:**

| Filter | Options |
|--------|---------|
| **Location** | Country, then Region/City within that country |
| **Category** | All, For Rent, For Sale |
| **Property Type — Residential** | Apartment, Villa, Duplex, Studio, Chalet, House, Penthouse, Townhouse, Other |
| **Property Type — Commercial** | Office, Shop, Warehouse, Other |
| **Property Type — Lands** | Other |
| **Bedrooms** | Number of bedrooms |
| **Bathrooms** | Number of bathrooms |
| **Price Range** | Adjustable slider from minimum to maximum price |
| **Area** | Adjustable slider from minimum to maximum area (in square meters) |

---

## 4. Property Listing Details

Each property page will display:

- **Title and description**
- **Photo gallery** — full-screen viewing with navigation between images
- **Price** and currency
- **Status** — For Sale, For Rent, Sold, or Under Offer
- **Reference number** — your existing reference numbers are preserved
- **Location** — country, region, and address
- **Specifications** — bedrooms, bathrooms, parking spaces, area (sqm), floor, year built
- **Features** — amenities such as pool, garden, garage, waterfront, view, furnished, etc.
- **Assigned agent** — name, photo, phone, and email
- **Contact form** — visitors can send an inquiry directly from the property page

---

## 5. User Accounts & Dashboard

Visitors can **create a free account** using their email address or by signing in with Google. Once registered, they can:

- **Submit property listings** — fill out a form with property details and upload photos; listings go through a review step before going live
- **Manage their properties** — edit or remove listings they have submitted
- **Save favorites** — bookmark properties they are interested in and view them from their personal dashboard

---

## 6. Admin Dashboard

As the website owner, you will have access to a **private control panel** that only administrators can see. This is where you manage everything behind the scenes:

- **Website overview** — see your total properties, agents, registered users, and recent inquiries at a glance
- **Add and manage properties** — create new listings directly, edit any existing listing, mark properties as featured, or remove them
- **Review submissions** — when registered users submit a property, you can approve or decline it before it goes live on the website
- **Manage agents** — add new agent profiles, update their information and photos, or remove agents who are no longer active
- **Contact inbox** — view every inquiry visitors have submitted through the website, and keep track of which ones you have responded to
- **Manage users** — see all registered users and assign roles (regular user, agent, or administrator)

---

## 7. Additional Features

- **WhatsApp button** — a floating WhatsApp icon on every page, linking directly to your number so visitors can reach you instantly
- **Social media links** — Instagram and Facebook icons in the website footer
- **Mobile-friendly design** — the website will look and work great on phones, tablets, and desktop computers
- **Contact form with notifications** — every inquiry submitted through the website is sent to you by email
- **Search engine optimized** — your properties and pages are structured so they appear in Google and other search engines

## 9. Delivery Phases

The project is organized into six phases. Each phase builds on the previous one, and you will have the opportunity to review progress at each milestone.

| Phase | Milestone | What's Included |
|-------|-----------|-----------------|
| **1 — Foundation** | Core website structure ready | Home page, About page, navigation menu, footer, user registration and login, all existing data migrated |
| **2 — Property Browsing** | Properties are browsable | Property listings page with full search and filters, individual property detail pages with photo gallery |
| **3 — User Features** | Registered users can manage listings | Dashboard, submit property form with photo uploads, manage own listings |
| **4 — Admin Dashboard** | Full management control panel | Property management, submission review, agent management, contact inbox, user management, website statistics |
| **5 — Engagement** | Visitor interaction features | Favorites system, contact form with email notifications, agents directory and profiles |
| **6 — Launch** | Website is live | Mobile responsiveness audit, search engine optimization, performance fine-tuning, deployment and go-live |
